01

The setup

The example portfolio resets to its target weights each trading day. Half of it sits in JPMorgan, Goldman Sachs and the XLF financial sector ETF. Apple and Microsoft make up another 30%, with the balance in long dated US Treasuries and gold.

This is a concentrated teaching example. It should not be read as a diversified portfolio recommendation.

02

Three ways to estimate the same loss threshold

At 95% confidence, each method estimates the one-day loss that should be exceeded on roughly one day in twenty.

Historical

$1,555

Uses the fifth percentile of the observed portfolio returns.

Parametric

$1,532

Fits a normal distribution using the sample mean and standard deviation.

Monte Carlo

$1,545

Samples 100,000 returns from that same fitted normal distribution.

03

Did the estimate hold up?

Each test day uses only the previous 252 returns. A breach occurs when the next loss is larger than the estimate.

Historical model

5.8%

47 breaches across 812 test days.

Kupiec p-value: 0.314

Parametric model

5.9%

48 breaches across 812 test days.

Kupiec p-value: 0.246

How to read this

Neither breach count is statistically inconsistent with a 5% rate at the 5% significance level. That is a narrow result. It does not prove that the models are accurate in every market regime.

06

What I would not conclude

A successful coverage test does not make VaR a worst-case loss estimate. The worst observed day lost 3.7%, more than twice the 95% VaR threshold. Losses beyond the threshold still matter.

The model also assumes that the next day resembles the selected history. Correlations and volatility can change quickly. The example rebalances every day, ignores costs and taxes, and contains overlapping financial exposure through two bank shares and XLF.

Method note

How the page is calculated

Adjusted closing prices come from Yahoo Finance through yfinance. The analysis uses simple daily returns from 4 January 2021 to 31 March 2025. Rolling backtests use a 252-day lookback and the Kupiec unconditional coverage test.
